PileMd

Think with you
PileMd is a Markdown Note App.
with Beautiful Markdown Editor and Comfy Notes.

Beautiful highlight

PileMd has really beautiful Markdown Editor, looks like Preview. It's like RichText Editor but actually Markdown Editor.
We think Markdown editor should not display both Editor and Preview at the one time. So we pursue the beauty of the Editor.

Comfortable editor

PileMd is really rich editor like word processor. It has Undo/Redo histories, Completing of lists and alse Opening links in editors. Of cause most of features have keymaps, so you can focus on editing.

Growth with you

Piled Markdown notes are your history, your growth, your own. The interface is created for you. To focus on writing and thinking.
It shows you Markdown notes as time-line based. We think people can find Markdown notes easily if it's on time-line, like note books in real life.

Pile keeping your think

If you don't want to lost your Markdown notes, just create Racks and/or Folders and drag and drop notes into it.
PileMd is based on philosophy of task management tools. So you can manage your notes you want.

Piled "NO"

We are obsessive to improve our life. PileMd has been built and broken again and again.
Tons of concept notes, various of proto-types, two production level apps. But we said "NO" to them, to create this Markdown note app. PileMd.
